Git常用指令

By 成华

Published 2020-01-03

基础指令

在当前目录初始化

git init

指定目录初始化

git init [name]

下载远程仓库的项目到本地

git clone [url]

配置

配置账户名

git config user.name [user_name]

配置邮箱

git config user.mail [user_mail]

配置远程地址

git config remote.url [remote_url]

添加删除文件

指定文件名添加

git add [file1] [file2] …

添加目录

git add [dir]

####添加所有文件

git add .

删指定文件

git rm [file1] [file2] …

代码提交

提交到仓库

git commit -m [message]

提交指定文件仓库

git commit [file1] [file2] … -m [message]

分支

罗列所有本地分支

git branch

列出所有远程分支

git branch -r

罗列所有分支

git branch -a

新建分支

git branch [branch-name]

新建并切换到分支

git checkout -b [branch]

新建分支并追远程分支

git branch –track [branch] [remote-branch]

切换分支

git checkout [branch-name]

来回切分支

git checkout -

建立追踪

git branch –set-upstream [branch] [remote-branch]

将指定分支合入当前分支

git merge [branch]

删除本地分支

git branch -d [branch-name]

删除远程分支

git push origin –delete [branch-name]

远程同步

获取远程所有变动

git fetch [remote]

新增远端仓库

git remote add [shortname] [url]

获取远程更新

git pull [remote] [branch]

####推送更新

git push [remote] [branch]

恢复

恢复提交

git checkout [file]

恢复所有提交

git checkout .

重置提交和当前工作区文件

git reset –hard

只重置提交到指定分支

git reset [commit]

重置提交及工作区文件

git reset –hard [commit]

查询信息

当前状态

git status

显示历史信息

git log

显示提交和工作区差异

git diff

显示某次提交的元数据和内容变化

git show [commit]

版本回退

日志查看

标签操作

创建标签

git tag tagName version

查看标签

git tag

查看标签详情

git show tagName

推送某个tag

git push origin v1.0

推送所有tag

git push origin –tags

组合提交指令

git
