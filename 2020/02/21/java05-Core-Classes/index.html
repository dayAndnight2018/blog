StringBuilder & String Buffer

String Buffer是线程安全的，内部采用了同步措施，StringBuilder是非同步的，效率高。

StringBuilder构造器

可以使用无参构造，或指定空间，或传入字符串。

默认的容量是16，超出大小再进行扩容。如果你预先知道空间占用，请预先指定容量。
