方法引用

这里的Apple :: getWeight 等价于 (Apple a) -> a.getWeight()

三种形式的方法引用

  1. 静态方法

    例如Integer.parseInt方法，可以直接使用 Integer :: parseInt

  2. 实例方法

    例如String的length()方法，可以直接使用 String :: length

  3. 实例的实例方法

    object :: getValue

2和3的区别在于，3用于在lambda表达式的内部引用外部对象的指定实例方法。
