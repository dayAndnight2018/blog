微服务基础概念

By 成华

Published 2021-01-09

本章的意义在于，提供基础的概念，以防后续存在不懂的概念需要查阅资料。

Spring

基于Spring的项目可以得到快速高效地构建，其提供了很多功能，如命名、依赖注入、数据访问、验证、国际化、切面编程等等。

  1. Spring Data提供了关系型和非关系型数据库的访问。

  2. Spring Batch提供了大数据量的处理支持。

  3. Spring Security提供了开箱即用的安全认证。

  4. Spring Cloud提供了分布式系统创建的功能。

  5. Spring Integration提供了不同模块之间交互的功能。

Spring Boot

Spring Boot提供了Spring项目的快速开发支持。

  1. 自动化配置。

  2. 开箱即用的starters。

  3. 不同环境切换的profile支持，支持不同环境下的配置切换。

我们来看一下一个POJO有多么复杂


我们可以看到，构造器、getters、setters、equals、hashCode、toString这些代码都是冗余的，完全可以被生成。

注解 含义
@AllArgsConstructor 生成包含所有字段的构造器
@FieldDefaults 让我们的字段成为private和final的
@Getter 生成getter方法
@ToString 生成toString方法
@EqualsAndHashCode 生成hashCode和equals方法
