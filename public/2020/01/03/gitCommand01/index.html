Git常用指令

By 成华

Published 2020-01-03

基础指令

初始化仓库


    git init

添加文件到缓冲区


    git add filename

添加所有文件到缓冲区


    git add .
    git add --all

清除缓冲区


    git rm -r --cached

删除文件


    git rm filename

提交备注


    git comment -m '备注内容'

注意：git add或git comment失败将不会提交。所以，comment必须有备注内容

查看状态


    git status

查看差异


    git diff filename

版本回退


    git reset
    git reset --hard HEAD^
    git reset --hard version

日志查看


    git log
    git reflog

分支管理

查看当前分支


    git branch

创建分支


    git branch branchName

切换分支


    git checkout branchName

创建并切换分支


    git checkout -b branchName

删除分支


    git branch -d branchName

合并某分支到当前分支


    git merge branchName

标签操作

创建标签


    git tag tagName version

查看标签


    git tag

查看标签详情


    git show tagName

推送某个tag


    git push origin v1.0

推送所有tag


    git push origin --tags

远端库相关

添加远端库


    git remote add origin git://xxx/abc.git

移除远端库


    git remote remove origin git://xxx/abc.git

推送到远端


    git push -u origin master

更新远端的内容到本地


    git pull

组合提交指令


    git rm -r --cached
    git add .
    git comment -m 'update'
    git push origin master
