Base Class

什么是base class?当我们定义了一个接口，这个接口定义了很多方法的时候，实现类需要实现全部方法，但是很多时候带有很多重复代码，因为没必要。因此，我们定义一个实现类实现这个接口，给出默认的实现方法。再由具体的类继承这个类去重写所需的方法，这个类就叫base class.
